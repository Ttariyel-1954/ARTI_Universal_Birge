ARTI Universal Birgə · v2.0 Təmiz

Tam istifadə təlimatı:
Mac açılışdan tam bəhrələnmə

Bu sənəd bölgə təmir-tikinti idarəetmə sisteminizi sıfırdan — Mac açıldığı andan, bütün əməliyyatları tam icra edənə qədər — addım-addım öyrədir. Hər komanda birbaşa kopyalayıb yapışdırmaq üçün hazırdır.

Fəsil 01 · Giriş

Layihə haqqında

ARTI Universal Birgə — Azərbaycan Respublikası Təhsil İnstitutunun 77 rayon üzrə məktəb təmir-tikinti layihələrini idarə etmək üçün enterprise-səviyyəli sistemidir. Sistem iki hissədən ibarətdir:

📊
Dashboard (Server)
Müdüriyyət üçün — bütün layihələrin idarəsi, statistika, xəritə, audit log
📱
Regional App (Mobile)
Bölgə işçiləri üçün — iPhone-dan yoxlama, şikayət, material əməliyyatları
🗄
PostgreSQL
Ortaq database — 9 schema, 40+ cədvəl, tam audit sistemi
🌐
Tailscale VPN
iPhone-dan təhlükəsiz uzaq qoşulma
✅
Bu sənəd standalone-dur Əlavə heç bir sənəd və ya internet axtarışı lazım deyil. Hər komanda kopyalamaq üçündür.

Fəsil 04 · Database

PostgreSQL-i işə salmaq

PostgreSQL işləyirmi?

PostgreSQL yoxlaması
pg_isready -h localhost -p 5432

Cavab nəticəsinə görə:

  • ✅ accepting connections → İŞLƏYİR — növbəti fəslə keçin
  • ❌ no response → aşağıdakı əmri icra edin

PostgreSQL-i işə salmaq

PostgreSQL işə salma
brew services start postgresql@18
sleep 5
pg_isready -h localhost -p 5432

Database statistikasını yoxlayın

Database məzmunu
psql arti_construction -P pager=off -c "SELECT (SELECT COUNT(*) FROM audit.users) AS istifadeci, (SELECT COUNT(*) FROM construction.projects) AS layihe, (SELECT COUNT(*) FROM inventory.materials) AS material, (SELECT COUNT(*) FROM geo.regions) AS rayon, (SELECT COUNT(*) FROM audit.query_log) AS audit_log;"

✅

Fəsil 06 · Regional App

Regional App-ı arxa planda işə salmaq

Regional app — iPhone-dakı bölgə işçilərinin qoşulduğu mobile interfeysdir. O, arxa planda işləyir, sizin RStudio ilə müdaxiləsinə ehtiyacınız yoxdur.

Regional app arxa planda başlat
cd ~/Desktop/ARTI_Universal_Birgə/platform/regional
nohup Rscript -e "options(shiny.port=3839); options(shiny.host='0.0.0.0'); shiny::runApp('.', launch.browser=FALSE)" > regional.log 2>&1 &
echo $! > regional.pid
sleep 10
tail -10 regional.log
Loading required package: shiny
✅ .env yükləndi
✅ API açarı yükləndi (108 simvol)
═══ DB Source: localhost (localhost:5432) ═══
Listening on http://0.0.0.0:3839

Regional-ın işlədiyini yoxla

Port 3839 yoxlaması
lsof -ti:3839 && echo "✅ Regional aktivdir" || echo "❌ Regional işləmir"

Brauzerdə test

Regional brauzerdə aç
open http://127.0.0.1:3839
⚠️
Regional RStudio-da AÇMAYın! Regional arxa planda işləyir — RStudio-da açmağa ehtiyac yoxdur. Port toqquşması olacaq.

Regional-ı dayandırmaq (iş günü sonunda)

Regional dayandır
cd ~/Desktop/ARTI_Universal_Birgə/platform/regional
kill $(cat regional.pid) 2>/dev/null
lsof -ti:3839 | xargs kill -9 2>/dev/null
echo "✅ Regional dayandırıldı"
